To compile our project simply run the command "javac -d .  @sources.txt".

From the same directory, run the command "java -D example.cpd.Store 224.0.0.0 3000 127.0.0.2 3002" to start a Store

From the same directory, run the command "java -D example.cpd.TestClient 127.0.0.1:3001 <operation> <add_args>" to start a TestClient
